

# 命令行终端聊天软件（字符界面） 

命名：CharLine 

关键词：单节点 IM， 最小可用化实例，cs 架构， 结构清晰，docker 部署



## 目标

单节点 IM + 学习 / 实验 / 自研项目



## 系统配置：

1. 2核 2G 的云服务器



## 基础想法：

1. client-server 架构 （单节点 IM ）

2. golang 实现

3. 配置文档化

4. 邀请制
   * server 创建（手动，或者自动）多个邀请码，每个邀请码只能使用一次 

5. server 不存储聊天信息

6. 简约的字符边框界面（可选）

7. client 端使用 sqlite 保存聊天记录，server 端不保存记录

8. server 使用 sqlite 做离线消息存储

9. jwt 用户登录验证
